## 🔧 Getting Started (My Setup)
To replicate my environment:
1.  Download the **Terraform amd64** binary.
2.  Install the **AWS CLI** and configure your credentials:
    ```bash
    aws configure
    ```
3.  Verify the connection:
    ```bash
    aws sts get-caller-identity
    ```
